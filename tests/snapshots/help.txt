Thin CLI over quire-rs (parse, extract, lookup, edit, validate, schema)

Usage: quire [OPTIONS] <COMMAND>

Commands:
  parse     Parse a markdown document to JSON
  extract   Extract structured records + edges from a document
  lookup    Look up one parsed section by heading, id, or block id
  edit      Edit one section/block of a document in place via byte-exact writeback
  validate  Validate a markdown document against its archetype structure
  schema    Emit an archetype's input contract (frontmatter schema + asserts) as JSON
  lint      Evaluate the module's advisory lint rules against a document
  fix       Surface (and with --write, apply) internal relative-path link fixes
  help      Print this message or the help of the given subcommand(s)

Options:
      --diagnostics-format <FORMAT>  Diagnostic stream format on stderr [default: human]
      --pretty                       Emit JSON output with pretty-printing where applicable
      --color <WHEN>                 Colorize human diagnostics on stderr: auto (TTY only, honours NO_COLOR), always, or never [default: auto]
  -h, --help                         Print help
  -V, --version                      Print version
